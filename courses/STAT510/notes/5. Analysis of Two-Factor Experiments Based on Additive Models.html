5. Analysis of Two-Factor Experiments Based on Additive Models

  • additive model: yijk+μ+αi+βj+ϵijk(i=1,2;j=1,2,3;k=1,2), factors do not interact

    • simple effect = main effect for additive model
    • test for main effects: α1=α2, β1=β2=β3, no main effect for first level ⇔α2=0, no main effect for second effect ⇔β2=β3=0
