15. ANOVA for Balanced Split-Plot Experiments

Model: yijk=μij+bk+wik+eijk, Genotype i=1,2,3, Fertilizer j=1,2,3,4, Block k=1,2,3,4.

  • Because the experiment is balanced, the GLS estimator is equal to the OLS estimator for any estimable Cβ: Cβ^Σ=Cβ^.

  • there are no terms in our model corresponding to Block×Fert.

  • E(MSBlocks×Fert)=E(MSBlocks×Geno×Fert)=σe2.

    E(MSGeno )=sbw−1∑i=1wE(y¯∗i..−y¯…)2=sbw−1∑∗i=1wE(μ¯∗i−μ¯..+w¯∗i−w¯..+e¯∗i.−e¯…)2=sb{∑∗i=1w(μ¯∗i−μ¯..)2w−1+E[∑∗i=1w(w¯∗i−w¯..)2w−1]+E[∑∗i=1w(e¯∗i..−e¯∗…..)2w−1]}=sb∑i=1w(μ¯∗i−μ¯..)2w−1+sbσ∗w2b+sbσe2sb=sb∑i=1w(μ¯∗i−μ¯.)2w−1+sσ∗w2+σe2

    E(MSBlock × Geno )=s(w−1)(b−1)∑i=1w∑k=1bE(y¯∗i⋅k−y¯∗i..−y¯∗⋅⋅k+y¯…)2=s(w−1)(b−1)∑∗i=1w∑k=1bE(wik−w¯∗i.−w¯∗⋅k+w¯∗..+e¯∗i⋅k−e¯∗i..−e¯∗..k+e¯∗…)2=s(w−1)(b−1)E[∑∗i=1w∑k=1b(wik−w¯∗i.)2−2∑∗i=1w∑k=1b(wik−w¯∗i)(w¯∗⋅k−w¯..)+∑i=1w∑k=1b(w¯∗⋅k−w¯..)2+e2 sum ]=s(w−1)(b−1)E[∑∗i=1w∑k=1b(wik−w¯∗i.)2−w∑∗k=1b(w¯∗⋅k−w¯..)2+e2sum]=s(w−1)(b−1)[w(b−1)σ∗w2−w(b−1)σw2/w+E(e2 sum )]=sσw2+σe2

    where

    E(e2sum)=E[∑i=1w∑k=1b(e¯i⋅k−e¯i.−e¯..k+e¯…)2]=(w−1)(b−1)sσe2

Inference for Whole-Plot

  • Test for whole-plot factor main effects:

    H0:μ¯1.=…=μ¯w.⇔H0:sbw−1∑i=1w(μ¯i.−μ¯..)2=0

    We compare MSGenoMSBlock×Geno to a central F-test with w−1 and (w−1)(b−1) degrees of freedom.

  • Comparison for whole-plot marginal means:

    H0:μ¯1.=μ¯2.

    Note that

    Var⁡(y¯∗1..−y¯∗2..)=Var⁡(μ¯∗1.−μ¯∗2.+w¯∗1.−w¯∗2.+e¯∗1..−e¯∗2⋅.)=2σw2b+2σe2sb=2sb(sσw2+σe2)=2sbE(MSBlock× Geno )

    so that Var^(y¯∗1..−y¯∗2..)=2sbMSBlock×Geno. We can use

    t=y¯∗1..−y¯∗2..−(μ¯∗1.−μ¯∗2.)2sbMSBlock × Geno ∼t(w−1)(b−1)

    to test H0.

  • Multiple test for whole-plot marginal means:

H0:C[μ¯∗1.⋮μ¯∗w.]=0,

we can use an F statistics with q and (w−1)(b−1) degrees of freedom:

F=(C[y¯∗1..⋮y¯∗w..])′[MSBlock ×GensbCC′]−1(C[y¯∗1..⋮y¯∗w..])q

where C is a matrix whose rows are contrast vectors so that C1=0.

Inference for Split-Plot

E(MSFert)=wbs−1∑j=1sE(y¯∗.j.−y¯∗...)2=wbs−1∑j=1sE(μ¯∗.j−μ¯∗..+e¯∗.j.−e¯∗...)2=wbs−1∑j=1s(μ¯∗.j−μ¯∗..)2+σe2E(MSerror)=σe2

  • Test for split-plot main effects:

    H0:μ¯∗.1=⋯=μ¯∗.s⟺H0:wbs−1∑j=1s(μ¯∗.j−μ¯∗..)2=0

    We compare MSFertMSerror to a central F distribution with s−1 and w(s−1)(b−1) degrees of freedom.

  • Comparison for split-plot marginal means:

    H0:μ¯.1=μ¯.2

    We can use

    t=y¯∗.1.−y¯∗.2.−(μ¯∗.1−μ¯∗.2)2wbMSError∼tw(s−1)(b−1)

    because Var⁡(y¯∗⋅1⋅−y¯∗⋅2⋅)=Var⁡(μ¯∗⋅1−μ¯∗⋅2+e¯∗⋅1⋅−e¯∗⋅2⋅)=2wbσe2=2wbE(MSError ).

  • Multiple test for split-plot marginal means:

    H0:C[μ¯∗.1⋮μ¯∗.w]=0,

    we can use an F statistics with q and w(s−1)(b−1) degrees of freedom:

    F=(C[y¯∗.1.⋮y¯∗.w.])′[MSErrorwbCC′]−1(C[y¯∗.1.⋮y¯∗.1.])q

    where C is a matrix whose rows are contrast vectors so that C1=0.

Inference for Interactions

E(MSGeno × Fert )=b(w−1)(s−1)∑i=1w∑j=1sE(y¯∗ij.−y¯∗i..−y¯∗.j.+y¯∗...)2=b(w−1)(s−1)∑i=1w∑j=1sE(μij−μ¯∗i.−μ¯∗.j+μ¯∗..+e¯∗ij.−e¯∗i..−e¯∗.j.+e¯∗...)2=⋯=b(w−1)(s−1)∑∗i=1w∑j=1s(μij−μ¯∗i.−μ¯∗.j+μ¯∗..)2+σ∗e2

  • Note that μij−μ¯∗i.−μ¯∗.j+μ¯∗..=0.∀i,j is equivalent to μ∗ij−μ¯∗i\*j−μ¯∗ij∗+μ¯i\*j\*,∀i≠i∗,j≠j∗. Therefore, b(w−1)(s−1)∑i=1w∑j=1s(μij−μ¯∗i.−μ¯∗.j+μ¯..)2=0 can be used as null hypothesis for no interactions between genotypes and fertilizers.

  • Test for Whole * Split interaction Effects

    H0:b(w−1)(s−1)∑i=1w∑j=1s(μij−μ¯∗i.−μ¯∗.j+μ¯..)2=0

    We can compare MSGeno×FertMSError to a central F distribution with (w−1)(s−1) and w(s−1)(b−1) degrees of freedom.

  • Inference for simple effects:

    H0:μ11=μ12

    Note that Var^(y¯11.−y¯12.)=2bMSerror, we can use t=y¯∗11⋅−y¯∗12⋅−(μ11−μ12)2bMSError ∼tw(s−1)(b−1) to test H0.

    However, Var(y¯11.−y¯21.)=2b(σw2+σe2) which is not a constant times any expected mean square from our ANOVA table. E(1sMSBlock×Geno+s−1sMSerror)=σw2+σe2s+(s−1)σe2s=σw2+σe2. So that Var^(y¯11.−y¯21.)=2sbMSBlock×Geno+2s−2sMSerror. We can use

    y¯∗11⋅−y¯∗21⋅−(μ11−μ12)2sbMSBlock×Geno+2s−2sMSerror∼td with d degrees of freedom

  • Inference for Cell Means μij: Note Var(y¯ij.)=σb2b+σw2b+σe2b so that we can construct unbiased estimator with approximate degrees of freedom from Cochran-Satterthwaite.
